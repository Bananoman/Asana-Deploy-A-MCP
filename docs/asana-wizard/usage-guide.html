Configuracion

Claude Code (.mcp.json)

{
  "mcpServers": {
    "asana": {
      "command": "node",
      "args": ["/ruta/a/Asana-Deploy-A-MCP/src/server.js"],
      "env": {
        "ASANA_TOKEN": "TOKEN_DEL_CLIENTE",
        "ASANA_TOOL_MODE": "efficient",
        "ASANA_RESPONSE_MODE": "compact"
      }
    }
  }
}

Tips para Consultores

Siempre empieza con assess_asana_maturity — te da el contexto que necesitas para todo lo demas.
Usa ASANA_RESPONSE_MODE=compact para reducir tokens 50-70% en respuestas.
Usa ASANA_READ_ONLY=true para explorar workspaces sin riesgo de modificar nada.
Valida con validate_ai_capability antes de generate_teammate_blueprint para no perder tiempo en Teammates inviables.
Un solo prompt para todo: asana_generate_deliverables genera template + DVA + ROI en un flujo.
Revisa get_asana_guide con topic="prebuilt_teammates" antes de crear Teammates custom — puede que ya exista uno prebuilt que sirve.
